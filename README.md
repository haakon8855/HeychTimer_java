# HeychTimer_java

HeychTimer is a Rubik's Cube timer for speedsolving made in in Java with JavaFX.

## Features

* The basic timer
* Inspection time (15s)
* A list with last solves
* 3x3 random scramble for each solve
* CSV-export with solvetime and scramble
